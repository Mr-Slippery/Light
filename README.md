# Light - Flashlight App

A simple Android flashlight app that turns your phone's flash into a flashlight.

## Features

- Single button to toggle flashlight on/off
- Simple, clean black interface
- Button turns yellow when flashlight is on
- Automatically turns off flashlight when app is paused or closed

## Requirements

- Android 6.0 (API 23) or higher
- Device with camera flash

## Building

1. Open the project in Android Studio
2. Build and run on your Android device

## Permissions

The app requires camera permission to access the device's flash.
